# Hotel Management Frontend

Фронтенд для системы управления отелем.

## Технологии

- React 18
- TypeScript
- Material-UI (MUI)
- React Router
- Axios
- Vite

## Установка

```bash
npm install
```

## Настройка

Создайте файл `.env` в корне проекта frontend (опционально):
```
VITE_API_URL=http://localhost:8080
```

По умолчанию используется `http://localhost:8080`

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу http://localhost:3000

## Сборка

```bash
npm run build
```

## Структура проекта

- `src/components/` - React компоненты
  - `admin/` - Компоненты админ-панели
  - `Login.tsx` - Страница входа
  - `AdminPanel.tsx` - Панель администратора
  - `ClientPanel.tsx` - Панель клиента
  - `EmployeePanel.tsx` - Панель работника
  - `Layout.tsx` - Общий layout с навигацией
  - `ProtectedRoute.tsx` - Защита маршрутов
- `src/services/` - API сервисы
- `src/contexts/` - React контексты (Auth)
- `src/types/` - TypeScript типы
- `src/utils/` - Утилиты

## Роли пользователей

- **ADMIN** - Полный доступ ко всем функциям управления:
  - Управление клиентами (CRUD, поиск по городу, создание счетов)
  - Управление номерами (CRUD, просмотр свободных номеров, жильцов)
  - Управление счетами (CRUD)
  - Управление сотрудниками (CRUD)
  - Управление расписанием уборки (CRUD)
  - Управление пользователями (CRUD)
  
- **CLIENT** - Просмотр своих счетов

- **WORKER** - Просмотр своего расписания уборки

## Особенности

- JWT аутентификация
- Автоматический редирект на страницу входа при истечении токена
- Защита маршрутов по ролям
- Полная реализация всех эндпоинтов бэкенда
- Удобный интерфейс с Material-UI
